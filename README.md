Controle de Gastos
Sistema simples de controle de gastos pessoais feito em Python, salvando os dados em um arquivo JSON.

Funcionalidades
Adicionar gastos com nome, valor e categoria
Listar todos os gastos cadastrados
Ver o total gasto
Filtrar gastos por categoria
Remover gastos pelo ID
Como usar
1. Clone o repositório

bash

git clone https://github.com/CodeShinu/controle-de-gastos.git
2. Entre na pasta

bash

cd controle-de-gastos
3. Rode o programa

bash

python gastos.py
